# Ultimate Copywriter & Marketer Skill Pack

This folder contains a comprehensive agent skill for copywriting, direct response, brand strategy, positioning, offer design, conversion optimization, content marketing, and marketing QA.

## Files

- `SKILL.md` — The core operating instructions. Load this first.
- `01_research_and_strategy.md` — Customer research, VOC mining, segmentation, market diagnosis, positioning, brand/category strategy.
- `02_offer_and_positioning.md` — Offer engineering, value propositions, pricing/packaging, risk reversal, proof, urgency, guarantees.
- `03_copywriting_playbook.md` — Headlines, leads, sales arguments, bullets, proof, CTAs, storytelling, email, ads, VSLs, landing pages.
- `04_asset_recipes_and_examples.md` — Practical recipes and sample copy for common assets.
- `05_brand_voice_and_content.md` — Voice systems, narrative, content strategy, SEO, thought leadership, social, community, editorial calendars.
- `06_conversion_qa_rubrics.md` — Review rubrics, anti-patterns, compliance guardrails, testing plans, measurement.
- `07_author_synthesis_matrix.md` — Operationalized principles from legendary and contemporary copywriting/marketing schools.
- `08_industry_playbooks.md` — Category-specific guidance and examples for SaaS, ecommerce, services, AI, nonprofit, local, finance, healthcare, developer tools, and more.
- `09_agent_prompt_templates.md` — Reusable prompts and workflows for agents.
- `10_full_funnel_campaign_blueprints.md` — Complete campaign systems and examples across funnel stages.
- `manifest.json` — Metadata for tooling.

## Intended use

Use this pack as the instruction corpus for a high-context AI agent. The main `SKILL.md` is intentionally self-contained enough to work on its own, while the supporting files provide deeper modules and examples.

The pack synthesizes generally known principles from classic and contemporary copywriting/marketing practice, but it does not reproduce copyrighted books, sales letters, ads, or proprietary course material verbatim.
